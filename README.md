## Laravel 10 - template
## Install

1. clone the repo
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan ide-helper:generate
6. php artisan sail:add
7. sail up -d
8. sail shell
9. php artisan migrate
10. php artisan db:seed
11. npm install
12. npm run dev / npm run build
